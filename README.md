# SBE Scheduler

This project is created with Larawind

## Requirements

-   Laravel installer
-   Composer
-   Npm installer

## Installation

```

# cd into your project directory
cd sbe-scheduler

#install composer and npm packages
composer install
npm install && npm run dev

# Start prepare the environment:
php artisan migrate
php artisan db:seed

# Run your server
php artisan serve
```
